Comandos para ejecutar el proyecto Waifuverse (Django)

1. Instalar dependencias (si hay requirements.txt):
   pip install -r requirements.txt

2. Aplicar migraciones de base de datos:
   python manage.py migrate

3. Crear superusuario (opcional, para acceder al admin):
   python manage.py createsuperuser

4. Ejecutar el servidor de desarrollo:
   python manage.py runserver

El servidor se ejecutará en http://127.0.0.1:8000/

Para detener el servidor, presiona Ctrl+C en la terminal.

Notas:
- Asegúrate de tener Python y Django instalados.
- Las imágenes de las waifus deben estar en la carpeta media/ dentro del directorio del proyecto.
- El proyecto incluye mejoras recientes: animaciones, modo oscuro, navegación mejorada y optimización de imágenes.
